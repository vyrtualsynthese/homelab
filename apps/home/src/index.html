Configure your hostfile :

To access homelab services your need to configure your host file.

How to edit your host file :

  • For windows
  • For OSX
  • For Linux

add those lines in your host file :

# Homelab
192.168.1.22    kub.local
192.168.1.22    home.local
192.168.1.22    dashy.local
192.168.1.22    grafana.local
192.168.1.22    n8n.local
